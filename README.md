## 📋 About

A simple and entertaining lottery game where users can pick numbers and check if they win. Built as a JavaScript DOM manipulation and logic practice project.

## ✨ Features

- Pick lottery numbers
- Randomly generate winning numbers
- Check if your numbers match the winning combination
- Visual feedback for wins and losses
- Responsive design

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/fuatsevinc/HTML-CSS-JS-LottaryGame.git
   ```

2. Open `index.html` in your browser.
